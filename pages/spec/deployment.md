### try_files
```nginx
server {
    listen       9999;
    server_name  localhost;
    location / {
        # root   html;
        root /static/projects/test_try_files/dist;
        # index index.html;
        try_files $uri $uri/ /index.html;
    }
}
```

| 应用场景       | 描述                       |
|--------------|----------------------------|
| 单页面应用 history模式降级     | 当文件不存在时返回自定义404页面（这个404返回应用的首页html即可），这样即可实现刷新之后，不会404，同时保留在当前路由 |
